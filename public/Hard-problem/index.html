Hard problem

12月 22, 2019 143

CodeForces 706C

題意:

輸入N(有多少字)、每個字顛倒的成本和每個字，問你不改變這串字的先後順序，藉由顛倒某些字來將這些字排成字典順序最少需要多少成本？

思路:

直接將所有字及所有顛倒後的字存起來，然後紀錄後面的字要滿足比前面的字大的最小成本，逐個更新就是答案。

newer Gennady and a Card Game
