# Hexo下的Markdown語法
### 標題
```
# 一級標題        //一級標題 對應 <h1> </h1>
## 二級標題       //二級標題 對應 <h2> </h2>
### 三級標題      //三級標題 對應 <h3> </h3>
#### 四級標題     //四級標題 對應 <h4> </h4>
##### 五級標題    //五級標題 對應 <h5> </h5>
###### 六級標題   //六級標題 對應 <h6> </h6>
####### 七級標題  //六級標題 對應 <h7> </h7>
```

### 文字強調

```
**文字** , __文字__                   //加粗
```

範例: **文字** , __文字__

```
*文字* , _文字_                       //斜體
```

範例: *文字* , _文字_  

```
***文字*** , __文字__ , *__文字__*    //加粗斜體
```

範例: ***文字*** , __文字__ , *__文字__*

```
~~文字~~                             //刪除
```

範例: ~~文字~~ 

```
<code>文字</code>    //文字Mark
```

範例: <code>文字</code>

### 表格
- 和 | 分割行和列
: 控制對齊方式

#### 無冒號對齊

```
   列1   | 列2 | 列3 
   ----- | --- | ---- 
第1行 | 777  | 777 
第2行 | 777  | 777  
第3行 | 777  | 777
```

範例:

   列1   | 列2 | 列3 
   ----- | --- | ---- 
第1行 | 777  | 777 
第2行 | 777  | 777  
第3行 | 777  | 777

#### 冒號對齊:

```
| 靠左    |  靠右 | 置中 |
| :-------- | -------:| :--: |
| Computer  | 5000 元 |  1台 |
| Phone     | 1999 元 |  1部 |
```

範例:

| 靠左       |  靠右   | 置中 |
| :-------- | -------:| :--: |
| Computer  | 5000 元 |  1台 |
| Phone     | 1999 元 |  1部 |
